# 📊 Customer Churn Analysis Dashboard

## 🔍 Project Overview
This project analyzes customer churn using Power BI to identify patterns and provide actionable business recommendations to improve customer retention.

---

## 📊 Dataset
- Source: Kaggle (Telco-style dataset)
- Total Records: 64,000+ customers
- Domain: Telecom / SaaS

---

## 🛠 Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Power Query (Data Cleaning)

---

## 📈 Key KPIs
- Total Customers
- Total Churned Customers
- Churn Rate (%)

---

## 📊 Dashboard Features
- Churn analysis by Gender
- Churn analysis by Subscription Type
- KPI cards for business overview
- Interactive dashboard design

---

## 🔍 Key Insights
- Churn rate is approximately 38–47%, indicating a major business issue
- Gender has minimal impact on churn
- Subscription type does not significantly affect churn
- Key drivers: customer experience factors like support issues and payment delays

---

## 💡 Business Recommendations
- Improve customer support and reduce resolution time
- Introduce onboarding programs for new customers
- Implement loyalty and retention strategies
- Reduce payment delays and improve billing experience
- Develop churn prediction models

---

- Power BI Dashboard (.pbix)

## 🎯 Conclusion
This project demonstrates how data analysis can be used to identify customer churn patterns and provide data-driven business solutions.
